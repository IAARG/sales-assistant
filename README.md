# 🧠 Sales Assistant – IA para Vendedores

Assistente de vendas com inteligência artificial que escuta conversas em tempo real e sugere argumentos, respostas a objeções e comparações de produtos para auxiliar vendedores durante negociações comerciais.

🔗 Acesse agora: [https://www.iaarg.com.br](https://www.iaarg.com.br)

---

## 🚀 Funcionalidades

- ✅ Login seguro com Google, GitHub ou email/senha via NextAuth
- ✅ Dashboard do usuário com nome, email e logout
- ✅ Transcrição de áudio em tempo real direto no navegador
- ✅ IA integrada que analisa o diálogo e fornece argumentos comerciais na hora
- ✅ Compatível com desktop e mobile
- ✅ Deploy completo na Vercel com domínio customizado

---

## 🛠️ Tecnologias Usadas

- **Next.js 14**
- **React + Tailwind CSS**
- **OpenAI GPT-4o** para argumentos de venda
- **Whisper (transcrição de voz)**
- **NextAuth.js** para autenticação
- **PlanetScale (MySQL)** para banco de dados
- **Vercel** para deploy
